Datadata API

datadata platform public api

Authentication​

  • API Key: ApiKeyAuth
  • OAuth 2.0: OAuth2AccessCode

API Key authentication. Prefix with "dd-" when passing the key.

Security Scheme Type:

apiKey

Header parameter name:

X-Datadata-Api-Key

OAuth2 Authorization Code flow

Security Scheme Type:

oauth2

OAuth Flow (authorizationCode):

Token URL: https://www.datadata.com/api/oauth2/token

Authorization URL: https://www.datadata.com/auth/oauth/authorize

Refresh URL: https://www.datadata.com/api/oauth2/token

Scopes:

  • datasources:read: Read datasource info, list tables, and describe table schemas

  • datasources:scan: Scan datasource schemas and manage table/column comments

  • datasources:replace-file: Replace file backing a file-type datasource

  • data-spaces:write: Create, describe, drop tables and insert rows in data spaces

  • executions:get: Get execution details and status

  • queries:execute: Execute saved queries

  • queries:execute-adhoc: Execute adhoc queries with inline scripts
