# 创建与配置图表

## 从查询创建图表

1. 在 Studio 中打开任意查询
2. 点击 **运行** 获取结果
3. 在结果上方点点击 **可视化** 按钮
4. 在弹出的面板中选择图表类型
5. 配置右侧面板中的维度和系列字段
6. 图表自动渲染预览
7. 点击 **保存** 将其添加到面板

## 图表配置面板

选择图表类型后，右侧会显示配置面板，主要包含以下设置：

### 维度 (Dimension)

选择 X 轴或类目字段，通常是日期、类别等。

### 系列 (Series)

每个系列对应一个数据字段，可以添加多个系列进行对比。每个系列可配置：

- **数据字段** — 选择数据集中的列
- **颜色** — 系列颜色
- **图表类型**（直角坐标系） — 可选 line / bar / area / scatter
- **Y 轴** — 绑定到左侧或右侧数值轴
- **标签** — 显示数据标签
- **标记线 (Mark Line)** — 添加参考线（如均值、最大值、阈值）
- **标记区域 (Mark Area)** — 添加高亮区域
- **条件颜色 (Dynamic Color)** — 根据数值动态改变颜色

### 图例 (Legend)

| 配置项 | 说明 |
| --- | --- |
| 显示/隐藏 | 控制图例可见性 |
| 方向 | 水平或垂直 |
| 位置 | 图例在图表中的位置 |
| 选择模式 | 支持单选或多选切换系列 |

### 坐标轴 (Axis)

类目轴和数值轴各有丰富的配置选项：

| 配置项 | 类目轴 | 数值轴 |
| --- | --- | --- |
| 名称 | ✓ | ✓ |
| 显示/隐藏 | ✓ | ✓ |
| 轴线和刻度线 | ✓ | ✓ |
| 分割线 | ✓ | ✓ |
| 标签旋转 | ✓ | — |
| 取值范围 | — | ✓ (最小值/最大值) |
| 缩放 | — | ✓ (Scale) |
| 小数位 | — | ✓ |

### 数据缩放 (DataZoom)

前后端数据缩放，用于大数据量的聚焦分析：

- **X 轴缩放** — 支持滑块选择时间范围
- **Y 轴缩放** — 支持滑块选择数值范围

### 堆叠 (Stack)

多个系列可以按堆叠方式显示：

- **全部堆叠** — 所有正值/负值叠加
- **同号堆叠** — 仅同符号堆叠
- **正/负值堆叠** — 仅正或仅负值堆叠

## 配置示例：折线图

```yaml
维度: trade_date
系列:
  - 字段: close
    颜色: #2563eb
    类型: line
  - 字段: high
    颜色: #ef4444
    类型: line
    标记线:
      - 标签: "最高价"
        值: 263.7
图例: 显示
X 轴: trade_date
数据缩放: 滑块开启
```

## K 线图配置

K 线图需要特定的 OHLC 字段映射：

| 字段 | 说明 |
| --- | --- |
| **timestamp** | 时间戳字段 |
| **openingPrice** | 开盘价 |
| **closingPrice** | 收盘价 |
| **highestPrice** | 最高价 |
| **lowestPrice** | 最低价 |
| **volume** | 成交量（可选） |
| **turnover** | 成交额（可选） |

K 线图支持的特性：

- **技术指标** — 内置指标或自定义指标叠加
- **子面板** — 多面板分割显示（如价格 + 成交量 + MACD）
- **蜡烛图样式** — 实心、空心、OHLC、Area 等
- **颜色风格** — 红涨绿跌 / 绿涨红跌
- **精度设置** — 价格精度和成交量精度

## 保存图表

完成配置后，点击 **保存**，选择要添加到哪个面板，即可将图表作为 widget 添加到仪表盘中。

## 面板布局

在面板中你可以拖拽调整每个 widget 的位置和大小，自由组合成监控仪表盘。
