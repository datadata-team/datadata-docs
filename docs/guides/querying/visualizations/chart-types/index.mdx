# 图表类型参考

Datadata 提供丰富的图表类型，覆盖统计分析、数据探索和金融量化三大类场景。

## 统计分析类

这类图表适用于通用数据分析和统计展示。

| 图表类型 | 分类 | 适用场景 |
| --- | --- | --- |
| **折线图** (Line) | 直角坐标系 | 趋势分析，时间序列数据 |
| **柱状图** (Bar) | 直角坐标系 | 分类对比，数值比较 |
| **面积图** (Area) | 直角坐标系 | 展示数值变化趋势和占比 |
| **水平柱状图** (Row) | 直角坐标系 | 横向对比，适合长标签 |
| **饼图** (Pie) | 占比 | 部分占整体的比例关系 |
| **环形图** (Doughnut) | 占比 | 类似饼图，中空可显示额外信息 |
| **南丁格尔玫瑰图** (Rose) | 占比 | 通过扇形角度和半径展示多维数据 |
| **散点图** (Scatter) | 直角坐标系 | 展示两个变量的相关性 |
| **表格** (Table) | 基础 | 原始数据展示，支持排序和列宽调整 |
| **指标卡** (Number) | 基础 | 单个关键指标的突出展示 |

## 数据探索类

这类图表用于发现数据中的模式和分布。

| 图表类型 | 分类 | 适用场景 |
| --- | --- | --- |
| **热力图** (Heatmap) | 矩阵 | 展示数据密度或二维分布 |
| **矩形树图** (Treemap) | 层级 | 展示层级结构和占比 |
| **日历热力图** (Calendar) | 时间 | 展示日期维度的数据分布 |
| **桑基图** (Sankey) | 流程 | 展示流量、转化路径 |
| **旭日图** (Sunburst) | 层级 | 多层级环形占比 |
| **漏斗图** (Funnel) | 流程 | 展示转化过程的递进减少 |
| **地图** (Map) | 地理 | 基于 Leaflet 的地理数据展示 |

## 金融量化类

这类图表专为金融和量化分析场景设计。

| 图表类型 | 说明 |
| --- | --- |
| **K 线图 / 蜡烛图** (Candlestick) | 基于 KlineCharts 库的金融 K 线图，支持 OHLC 数据、指标叠加和子面板 |
| **实盘回放** (Live Replay) | 交易策略实盘回放模拟，用于策略验证和复盘分析 |

## 直角坐标系图表配置

折线图、柱状图、面积图、水平柱状图和散点图共享直角坐标系配置：

- **维度字段 (dimension)** — X 轴字段
- **系列 (series)** — 每个系列可配置：
  - 数据字段、颜色、标签
  - 数据缩放 (DataZoom)
  - 标记线和标记区域
  - 条件颜色
- **双数值轴** — 支持左右两侧独立的 Y 轴
- **堆叠** — 支持数值堆叠显示
- **图例** — 支持位置、方向、选择模式

## 图表事件与联动

图表支持钻取和联动事件，可用于面板中各图表之间的交互。
