# 数据可视化

查询执行完成后，你可以将结果转换为各种专业图表进行可视化分析。

## 创建图表

1. 在查询编辑器中点击 **运行** 获取数据
2. 在结果上方点点击 **可视化** 按钮
3. 选择图表类型
4. 配置维度和指标字段
5. 图表自动渲染预览
6. 点击 **保存** 可将图表保存到面板

查询结果上方还提供了快捷图表按钮：

- **Query Results** — 表格视图
- **蜡烛图** — 直接渲染 OHLC K 线图
- **可视化** — 打开图表类型选择器

## 图表配置

每个图表可以配置维度（X 轴）、系列（数据字段）、颜色、坐标轴、图例和标记等。详情请参阅：

- [图表类型参考](/docs/guides/querying/visualizations/chart-types) — 所有 18 种图表类型
- [创建与配置图表](/docs/guides/querying/visualizations/creating-charts) — 详细配置说明

## 保存到面板

配置好的图表可以保存到 **Dashboard（面板）** 中，与其他图表和媒体卡片组合成一个完整的监控看板。
