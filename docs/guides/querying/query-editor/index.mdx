# 查询编辑器

查询编辑器是 Datadata 的核心交互界面，集成了 SQL/DQL 编辑、数据源选取、结果预览和 AI 辅助功能。

## 界面布局

编辑器界面分为三个区域：

```
┌──────────────────────────────────────────────────┐
│  左侧面板：数据源选择器                            │
│  ┌─────────────────┐                             │
│  │ 已绑定的数据源    │     编辑器区域               │
│  │ 热门数据源       │  ┌──────────────────────┐   │
│  │   crypto tick    │  │ SQL / DQL 编辑器      │   │
│  │ 我的数据源       │  │                       │   │
│  │   Customers      │  │                       │   │
│  │   json 测试      │  └──────────────────────┘   │
│  │ 测试             │                             │
│  └─────────────────┘                             │
│                              AI Copilot 面板      │
│                              ┌──────────────┐     │
│                              │ AI 助手       │     │
│                              │ · 分析查询    │     │
│                              │ · 建议改进    │     │
│                              └──────────────┘     │
└──────────────────────────────────────────────────┘
```

## 数据源选择

左侧数据源面板分为三组：

- **已绑定的数据源** — 当前查询绑定的数据源
- **热门数据源** — 社区中常用的公开数据源（如 crypto tick）
- **我的数据源** — 你创建或上传的数据源

点击数据源名称即可将其绑定到当前查询。

## SQL / DQL 模式切换

编辑器顶部有一个模式选择器，可在 **SQL** 和 **DQL** 之间切换：

- **SQL 模式**：适合编写标准 SQL 查询语句
- **DQL 模式**：适合编写基于 Starlark 的数据处理脚本

## AI Copilot

右侧 AI Copilot 面板可以：

- 分析当前查询并提供优化建议
- 回答关于数据的问题
- 帮助理解数据源的价值

内置快捷提问：

> "介绍一下你自己"
> "当前 Query 的数据源有哪些分析价值？"
> "我的这个 SQL 写的怎么样？"

## 保存与分享

- 点击 **保存** 按钮保存当前查询
- 查询可见性可设为 **public**（公开）或 **private**（私有）
- 公开查询会出现在 Explore 页面供他人参考
- 支持添加标签（如 `#黄金`、`#tick`）方便检索
