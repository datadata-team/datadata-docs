# 数据查询

Datadata 支持两种查询方式：**标准 SQL** 和 **DQL 脚本**。

## 工作流程

```
选择数据源 → 编写查询脚本 → 运行 → 查看结果 → 可视化
```

## 查询模式

| 模式 | 适用场景 | 特点 |
| --- | --- | --- |
| **SQL** | 日常数据探索与分析 | 标准 SQL 语法，支持 WHERE、JOIN、GROUP BY 等子句 |
| **DQL** | 复杂数据处理流水线 | 基于 Starlark 的扩展脚本，DataFrame 操作、HTTP 请求等 |

DQL 是 Datadata 的特色能力，支持丰富的数据处理函数，详见 [DQL 查询脚本](/docs/guides/dql/) 章节。

## 查询引擎

- **DuckDB** — 默认引擎，支持跨数据源关联查询
- **ClickHouse** — 高性能时序引擎，适合大数据量分析
