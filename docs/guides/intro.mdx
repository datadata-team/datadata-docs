# Datadata 平台简介

Datadata 是一个 AI 驱动的数据平台，专注于让数据分析更**精准高效**。它提供完整的端到端数据分析工作流：

## 核心功能

- **数据查询** — 使用标准 SQL 或 DQL 脚本探索和分析数据，支持跨数据源关联查询 (DuckDB + ClickHouse)
- **可视化面板** — 将查询结果组合为仪表盘，使用十余种专业图表类型进行监控与洞察
- **AI Skills** — 通过 MCP、CLI 或 DQL 脚本让 AI Agent 与平台直接交互
- **数据源管理** — 连接外部数据库、上传数据文件或创建数据空间，统一管理数据资产
- **社区与洞察** — 发布数据分析文章、分享查询与数据源，与社区共同探索数据价值
