# 面板 (Dashboard)

Dashboard 是一个可自由组合的内容看板。你可以将多个 **可视化图表（Visualization）** 以及 **媒体卡片** 组合到一个页面中，通过拖拽自由布局。

## Dashboard 与 Query 的区别

| | Query（查询） | Dashboard（面板） |
| --- | --- | --- |
| **用途** | 编写脚本、执行查询、查看结果 | 组合展示多个图表和内容 |
| **内容** | 一个查询 + 一个可视化 | 多个可视化 + 媒体卡 |
| **布局** | 固定 | 自由拖拽布局 |
| **场景** | 数据探索与分析 | 监控看板、报告展示 |

## 创建面板

1. 在 Studio 侧边栏点击 **面板**
2. 点击 **新建面板**
3. 创建空白面板
4. 点击 **添加 Widget** 选择组件

## 支持的 Widget 类型

| 类型 | 说明 |
| --- | --- |
| **可视化 (Visualization)** | 从已有查询中选择并嵌入图表 |
| **Markdown** | 使用 Markdown 编写说明文字 |
| **嵌入式视频** | 嵌入 Bilibili、YouTube 等视频 |
| **更多媒体卡片** | 其他富媒体内容 |

Dashboard 创建完成后，你可以在页面中拖拽调整各个 Widget 的位置和大小。

## 面板管理

- 面板列表支持按文件夹组织
- public / private 可见性设置
- 搜索和排序
