# AI Integration

Datadata 提供多组 AI Skills，让 AI Agent 能够通过自然语言与平台交互。

所有 Skills 可通过一条命令安装：

```bash
npx skills add datadata-team/datadata-skills
```

## Skills 列表

| Skill | 用途 |
| --- | --- |
| [datadata-mcp](/docs/guides/ai-skills/mcp-server/) | 指导 AI Agent 使用 Datadata MCP Server 查询和管理数据 |
| [datadata-api](/docs/guides/ai-skills/datadata-api/) | API 调用封装，用于 SQL 查询和 Data Spaces 管理 |
| [datadata-dql](/docs/guides/ai-skills/datadata-dql/) | DQL 脚本编写，用于复杂数据处理 |
| [datadata-memory](/docs/guides/ai-skills/datadata-memory/) | AI 跨会话持久化记忆管理 |

## DQL 语言参考

DQL 的完整语言参考详见独立的 [DQL 查询脚本](/docs/guides/dql/) 章节。
