# 上传数据文件

Datadata 支持上传 CSV、Excel、JSON 和 Parquet 文件作为数据源。

## 支持的文件格式

| 格式 | 说明 | 典型场景 |
| --- | --- | --- |
| **CSV** | 逗号分隔的表格数据 | 导出数据、数据交换 |
| **Excel** (.xlsx) | Microsoft Excel 电子表格 | 业务数据、报表 |
| **JSON** | 结构化 JSON 数据 | API 数据、配置数据 |
| **Parquet** | 列式存储格式 | 大数据分析 |

## 操作步骤

1. 在 Studio 侧边栏点击 **数据源**
2. 点击 **新建数据源** → **上传数据文件**
3. 选择文件类型（CSV / Excel / JSON / Parquet）
4. 上传文件或通过以下方式导入：
   - **本地上传** — 从本机选择文件
   - **S3 存储** — 从 Amazon S3 导入
   - **外部 URL** — 通过 URL 加载远程文件
5. 填写数据源名称
6. 点击 **创建**

## 本地文件上传

直接从本机选择文件上传，支持拖拽或点击选择。

## S3 存储导入

适用于已有 AWS S3 存储的数据文件，填写文件的 S3 Key 即可。

## 外部 URL 导入

通过 HTTP/HTTPS URL 直接加载远程数据文件，适用于公开数据集或 API 导出的数据。

## 创建后

文件上传完成后，Datadata 会自动解析文件结构并创建对应的数据源。你可以在查询中像使用普通数据源一样查询文件数据。
