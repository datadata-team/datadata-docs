# 数据源管理

Datadata 支持连接外部数据库、上传数据文件、创建数据空间三种方式接入数据。数据源是查询和分析的基础。

## 支持的数据源类型

| 类型 | 接入方式 | 典型用途 |
| --- | --- | --- |
| **MySQL** | 连接 | 业务数据库 |
| **Postgres** | 连接 | 业务数据库 |
| **ClickHouse** | 连接 | 高性能时序数据分析 |
| **DuckDB** | 连接 | 嵌入式分析 |
| **CSV** | 上传 / S3 / 外部 URL | 表格数据 |
| **Excel** | 上传 / S3 / 外部 URL | 电子表格 |
| **JSON** | 上传 / S3 / 外部 URL | 结构化数据 |
| **Parquet** | 上传 / S3 / 外部 URL | 列式存储数据 |
| **DuckLake (数据空间)** | 新建 | 平台托管的数据存储 |

## 三种接入方式

| 方式 | 说明 |
| --- | --- |
| **连接数据源** | 填写远程数据库的连接信息，直接接入外部数据库 |
| **上传数据文件** | 上传 CSV / Excel / JSON / Parquet 文件 |
| **新建数据空间** | 在平台托管的数据空间中创建表、写入数据 |

## 数据源管理功能

在 Studio 侧边栏点击 **数据源** 即可进入管理页面：

- **类型过滤** — 按数据库类型快速筛选
- **文件夹** — 用文件夹组织数据源
- **搜索** — 按名称搜索

点击某个数据源进入详情页后可以：

- 查看表结构（Schema）
- 触发 Schema 扫描
- 编辑基础信息（名称、描述、标签）
- 编辑连接配置
- 删除数据源

## 安全建议

连接外部数据库时，请遵守**最小权限原则**——仅为查询场景创建只读账户，切勿授予不必要的写入或管理员权限。
