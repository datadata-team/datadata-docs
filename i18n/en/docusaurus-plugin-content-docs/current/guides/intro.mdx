# Datadata Platform Overview

Datadata is an AI-driven data platform focused on making data analysis **precise and efficient**. It provides a complete end-to-end data analysis workflow:

## Core Features

- **Data Querying** — Explore and analyze data using standard SQL or DQL scripts, with cross-datasource JOIN support (DuckDB + ClickHouse)
- **Visual Dashboards** — Compose query results into monitoring dashboards with over a dozen chart types
- **AI Skills** — Enable AI agents to interact with the platform through MCP, CLI, or DQL scripts
- **Datasource Management** — Connect external databases, upload data files, or create Data Spaces for unified data asset management
- **Community & Insights** — Publish analysis articles, share queries and datasources with the community
