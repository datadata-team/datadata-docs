# AI Integration

Datadata provides multiple AI Skills that allow AI agents to interact with the platform through natural language.

All Skills can be installed with a single command:

```bash
npx skills add datadata-team/datadata-skills
```

## Skills List

| Skill | Purpose |
| --- | --- |
| [datadata-manual](/docs/guides/ai-skills/datadata-manual/) | Guide AI agents in interactive querying and data management via MCP Server |
| [datadata-rest-api](/docs/guides/ai-skills/datadata-rest-api/) | REST API reference for generating standalone Python scripts (crawlers, ETL, batch) |
| [datadata-dql](/docs/guides/ai-skills/datadata-dql/) | DQL scripting for complex data processing |
| [datadata-memory](/docs/guides/ai-skills/datadata-memory/) | Cross-session persistent memory management for AI |
