# Visual Dashboards

Visual dashboards let you transform query results into professional charts and compose them into monitoring dashboards.

## Workflow

```
Write a query → Run it → Click "Visualize" → Choose chart type → Configure → Save to dashboard
```

## Creating Charts from Queries

1. Open a query in Studio
2. Click **Run** to get results
3. Click the **Visualize** button above the results
4. Select a chart type from the picker
5. Configure the dimension and series fields
6. The chart renders in real-time preview
7. Click **Save** to add it to a dashboard

Quick chart buttons are also available above results:

- **Query Results** — Table view
- **Candlestick** — Direct OHLC K-line rendering

## Dashboard Management

Click **Dashboards** in the Studio sidebar:

- Click **New Dashboard** to create a blank dashboard
- Add existing charts to the dashboard
- Public / private visibility settings
