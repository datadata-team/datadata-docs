# Data Querying

Datadata supports two query modes: **standard SQL** and **DQL scripts**.

## Query Modes

| Mode | Use Case | Features |
| --- | --- | --- |
| **SQL** | Everyday data exploration | Standard SQL syntax with WHERE, JOIN, GROUP BY support |
| **DQL** | Complex data pipelines | Starlark-based scripting with DataFrame operations, HTTP requests |

## Query Engines

- **DuckDB** — Default engine, supports cross-datasource JOIN queries
- **ClickHouse** — High-performance time-series engine for large datasets
