# Data Visualization

After executing a query, you can transform the results into various professional charts for visual analysis.

## Creating Charts

1. Click **Run** in the query editor to get results
2. Click the **Visualize** button above the results
3. Select a chart type
4. Configure dimension and metric fields
5. The chart previews automatically
6. Click **Save** to add the chart to a dashboard

Quick chart buttons are also available above query results:

- **Query Results** — Table view
- **Candlestick** — Render OHLC K-line directly
- **Visualize** — Open the chart type selector

## Chart Configuration

Each chart can be configured with dimensions (X-axis), series (data fields), colors, axes, legends, and markers. For details, see:

- [Chart Type Reference](/docs/guides/querying/visualizations/chart-types) — All 18 chart types
- [Creating & Configuring Charts](/docs/guides/querying/visualizations/creating-charts) — Detailed configuration guide

## Saving to Dashboard

Configured charts can be saved to a **Dashboard**, where they can be combined with other charts and media cards to form a complete monitoring view.
