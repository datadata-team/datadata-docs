快速开始

现在，让我们来模拟获取您创建的 Charts，即通过 GET 方法请求 /api/v1/charts。

在 概览/身份认证 章节中，我们已经详细了解了如何获取 API-Token。

现在准备好您的 API-Token ，从下面的方法中任选一种，立即尝试一下吧。

  • Curl
  • Node
curl https://www.datadata.cn/api/v1/charts \
  -H 'x-datadata-api-token: <your-token>'
const response = await fetch('https://www.datadata.cn/api/v1/charts', {
  headers: {
    'x-datadata-api-token': '<your token>'
  }
})
const data = await response.json();
console.log(data);
